#### 85 product 案例

```python
def product(*args, repeat=1):
    pools = [tuple(pool) for pool in args] * repeat
    result = [[]]
    for pool in pools:
        result = [x+[y] for x in result for y in pool]
    for prod in result:
        yield tuple(prod)
```


调用函数：

```python
rtn = product('xyz', '12', repeat=3)
print(list(rtn))
```
