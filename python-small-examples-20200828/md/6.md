#### 6 判断是真是假　　

测试一个对象是True, 还是False.
```python
In [9]: bool([0,0,0])                                                           
Out[9]: True

In [10]: bool([])                                                               
Out[10]: False

In [11]: bool([1,0,1])                                                          
Out[11]: True
```
