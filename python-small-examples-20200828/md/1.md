#### 1 实现 relu

在神经网络中，`relu`作为神经元的激活函数：

```python
def relu(x):
    """
    x: 输入参数
    return：输出relu值
    """
    return max(0,x)                                                                 
```

测试：

```python
relu(5) # 5

relu(-1) # 0
```
